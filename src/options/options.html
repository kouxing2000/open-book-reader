How to use & shortcuts

Open it on any page — three ways:

  • Toolbar icon — smart pick: the 📖 reader for articles, the 🖼 gallery for image-heavy pages.
  • Right-click a page or image → Open in Book Reader → Auto / Reader / Gallery.
  • Keyboard: Alt+B reader · Alt+Shift+B gallery.

Want a site to always open one way? Right-click → Always open this site as Reader / Gallery, or add a path/wildcard pattern under Per-site rules below.

While reading (📖 reader)

  • ← → flip pages — or click the left / right edge of the book
  • Home / End first / last spread · Esc close
  • Topbar: ⊞ columns (2/3/4) · theme · A− / A+ font size · 🖼 switch to gallery

In the gallery (🖼 images)

  • Click a tile for the lightbox — ← → browse · Esc close
  • ▶ Auto-scroll hands-free browsing — A toggle · + / − speed
  • ⟳ Load all pulls lazy images · select tiles to download or ZIP · 📖 switch to reader

Change the keyboard shortcuts

Chrome controls these — they can't be edited inside the extension. Open Chrome's shortcut editor to set your own.

📖 Reader

Saved content picks When the reader grabs the wrong block, the ⌖ Pick button lets you choose the real content and Save for this site. Saved picks are listed here and reused automatically on that site. You can edit the CSS selector to fine-tune it: a class like .article-body is more robust than the auto-generated path, and a selector that matches several blocks (e.g. .intro, .post-body) reads them all. The ✓/✗ marks whether the selector is valid. Remove one to go back to automatic extraction.

Per-site rules Force a mode on specific sites instead of the smart auto-pick. The pattern matches host + path with * as a wildcard — example.com (whole site), example.com/blog/* (a section), *.example.com/* (all subdomains). When several match, the most specific wins. Applies to the toolbar icon and right-click menu; Alt+B / Alt+Shift+B always honor their own mode.

Privacy: collects no data and sends nothing to the developer. It runs on a page only when you invoke it (toolbar icon, shortcut, or right-click menu). It makes network requests only when you explicitly download images from the gallery.

Open source: this extension is fully open — read every line, verify the privacy claims, file an issue, or contribute at github.com/kouxing2000/open-book-reader.
